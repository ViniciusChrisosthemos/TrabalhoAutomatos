Descrição:
O trabalho final da disciplina de Linguagens Formais e Autômatos pode ser resumido assim: Cria um conversor de Autômatos Finitos Não-Deterministicos (AFND) para Deterministicos (AFD) e elaborar uma linguagem para usar como entrada.
O conversor foi feito em Python, portanto, para utiliza-lo precisa ter instalado em sua máquina um interpretador de Python 3 ou equivalente.

Inicialização:
No diretório raiz, digite:
	
	python main.py

Comandos:
	- lf   : Carrega um arquivo com o autômato
	- vw   : Valida uma palavra no autômato carregado
	- vws  : Valida um arquivo de palavras no autômato carregado
	- c    : Converte o autômato carregado em um AFD
	- p    : Exibe as informações do autômato
	- exit : Fecha o programa
	- help : Exibe os comandos na tela
